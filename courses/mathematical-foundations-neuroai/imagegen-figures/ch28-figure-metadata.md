# Chapter 28 Figure Metadata

Workers should not generate or include these images directly. These are proposed prompts for a later image-generation pass.

## Prompt 1: Time-Series Representations

Use case: scientific-educational. Asset type: lecture-note raster infographic. 16:10 horizontal composition for PDF. Show the same neural recording represented as event times, binned counts, a smoothed rate trace, and a multichannel trial-by-time matrix. Use sparse labels only: "events", "bins", "smoothed rate", "trials x time x channels". Equations should appear in the caption, not in the image. Avoid logos, watermarks, decorative stock imagery, and pseudo-data presented as real data.

## Prompt 2: DFT Basis And Spectrum

Use case: scientific-educational. Asset type: lecture-note raster infographic. 16:10 horizontal composition for PDF. Show a short time signal decomposed into low, medium, and high sinusoidal basis components, then a simple amplitude/power spectrum with one highlighted peak. Sparse in-image labels only: "time signal", "basis waves", "spectrum", "phase". Exact formulas in caption only. No logos, watermarks, decorative stock imagery, or pseudo-data presented as real data.

## Prompt 3: Linear Filter Impulse And Frequency Response

Use case: scientific-educational. Asset type: lecture-note raster infographic. 16:10 horizontal composition for PDF. Show an impulse response kernel, convolution smoothing of a noisy trace, and a frequency response attenuating high frequencies. Include a small causal versus centered-window comparison. Sparse labels only. Equations in caption only. Avoid logos, watermarks, decorative stock imagery, and pseudo-data presented as real data.

## Prompt 4: Neural Signal Modalities

Use case: scientific-educational. Asset type: lecture-note raster infographic. 16:10 horizontal composition for PDF. Show simplified panels for spikes, membrane voltage, LFP/EEG, calcium imaging, and fMRI arranged by temporal scale. Sparse labels only: modality names and relative time scale markers. Equations in caption only. No logos, watermarks, decorative stock imagery, or pseudo-data presented as real data.
