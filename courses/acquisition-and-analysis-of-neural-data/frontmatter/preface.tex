\chapter*{Preface}
\addcontentsline{toc}{chapter}{Preface}

These notes collect the mathematical and practical ideas needed to turn neural recordings into interpretable data objects. The emphasis is on spike trains, trial structure, summary statistics, and the assumptions that connect raw measurements to models of neural coding.

The notes are organized so that conceptual exposition, derivations, and reproducible analysis decisions stay in the same place. Code can reproduce figures and explore examples, but the notes state the definitions, conventions, and interpretation rules that make those analyses scientifically meaningful.

The first chapter sets the workflow: start from a recording, choose a response representation, quantify structure with simple statistics, and only then fit richer encoding or decoding models. This order is intentional. In neural data analysis, a poorly defined response variable cannot be rescued by a sophisticated downstream model.
